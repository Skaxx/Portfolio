A B O U T   M E

My fascination with IT world and computer hardware has been going on for many years. This is a passion that made me to take the first step towards it.

Starting from the basics, I learned the secrets of {HTML} & {CSS}, which allowed me to create my first websites at the age of 14/15. I liked the effects so much that I became more and more interested in this world.

Then, after a few years, it was time to learn the first framework {Bootstrap}, and expanded my knowledge of styles using {SCSS}, and acquire basic programming skills in {JavaScript} this allowed me to create more modern projects and learning and applying RWD and Clean Code techniques - it can be said that I have been programming regularly since 2020 and constantly expanding my knowledge.

Today, my current work is based on cooperation with AndyTrans Sp. z o.o. - where in 2022 I created a website, and I am currently working on the "LMS - AndyTrans" project - it is a web application aimed at improving the company's logistics activities, for more information please visit the subpage on commercial projects .

Currently, I am constantly developing and expanding my knowledge, so in the future I would like to be able to take an active part in creating the new generation of the Internet, which is {Web3} - this is a very interesting topic for me, similar to the {Blockchain architecture}, and therefore, creating "dApps" & "DeFi" applications and others that can have a positive impact on the future and security of the Internet.
